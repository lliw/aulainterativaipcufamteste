> Definição de Estruturas Condicionais

Uma Estrutura Condicional, é uma estrutura de programação que permite a alteração do fluxo de execução de um código através de uma condição. Ou seja, existe parte do código que só vai ser executado se determinada condição for verdadeia ou falsa.

> Definição de Estruturas Condicionais

Uma Estrutura Condicional, é uma estrutura de programação que permite a alteração do fluxo de execução de um código através de uma condição. Ou seja, existe parte do código que só vai ser executado se determinada condição for verdadeia ou falsa.

O fluxo de execução de uma Estrutura Condicional possui uma ou mais bifurcações.


> Condição Verdadeira e Falsa

O resultado de uma condição pode ser verdadeira ou falsa. E isso define o percurso de execução do código. Por exemplo, imagine um situação em que alguém está dirigindo um carro que pode estar ou não estar quebrado

SE o carro estiver quebrado:
você deve passar numa oficina antes de ir para casa
SENÃO:
você pode ir direto pra casa


Você decide, o carro está quebrado, ou não está quebrado?

> Condição Verdadeira e Falsa

O resultado de uma condição pode ser verdadeira ou falsa. E isso define o percurso de execução do código. Por exemplo, imagine um situação em que alguém está dirigindo um carro que pode estar ou não estar quebrado

SE o carro estiver quebrado:

Se o não carro está quebrado, a condição é falsa e ele pode ir direto pra casa.


> Formato das Estruturas Condicionais 

if (condição) :
		#Ações caso a condição seja verdadeira
	else:
		#Ações caso a condição seja falsa

O código do exemplo acima mostra a execução em conjunto das linhas:
, .

O início da estrutura condicional é denotada pelo comando if seguido de uma condição e do sinal de dois pontos (:). O "if" vem do inglês e significa "se", no sentido de: "se essa condiçao é verdadeira, então faça isso".

Depois vêm os comandos e operações que devem ser realizados caso a condição seja verdadeira. Note que existe um recuo na linha após o comando if, isso indica que essa linha está subordinada ao if.


> Formato das Estruturas Condicionais 

if (condição) :
		#Ações caso a condição seja verdadeira
	else:
		#Ações caso a condição seja falsa

O código do exemplo acima mostra a execução em conjunto das linhas:
, .

Depois vem o comando else também seguido de um sinal de dois pontos. O "else" vem do inglês e significa "senão" no sentido de "se alguma coisa, então faça isso. Senão, faça aquilo".

Este comando é facultativo, ele é usado caso haja ações se a condição for falsa. E da mesma forma que antes, a linha seguinte precisa de recuo.

> Operadores de Comparação

Operadores de comparação são utilizados para estabelecer uma relação de comparação entre valores numéricos. Decore os operadores mostrados da tabela abaixo:

Operador Operação Exemplo
Verdadeiro Falso
== Igual a 3 == 3 20 == 18
> Menor que 5 > 4 10 > 11
< Menor que 3 < 6 9 < 7
>= Maior ou igual 4 >= 4 3 >= 5
<= Menor ou igual 3 <= 5 7 <= 1
!= Diferente de 8 != 9 2 != 2
Observe que quando a condição do operador é satisfeita, a comparação é verdadeira. E quando a condição não é satisfeita, a comparação é falsa.


> Diferença entre "=" e "=="

Os símbolos = e == fazem parte da sintaxe do Python. Mas eles tem significados diferentes. Um deles é usado para atribuição e o outro para comparação.

No código a baixo, vemos os dois símbolos. Marque na tabela se o símbolo em destaque na linha indicada é de atribuição ou comparação. 

a = "a"
if ("a" == a):
	print("a igual a:",a)

linha 2 atribuição comparação

O símbolos = é usado para atribuição.
