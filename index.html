O que são variáveis?


Na matemática, a variável é uma incógnita que pode assumir o valor de qualquer elemento de um conjunto.

Nos algoritmos, a variável é um elemento que pode assumir um único valor de cada vez, que muda ao longo da execução do programa.

E, para a memória do computador, a variável é uma região da memória que contém um valor e é apontada por um identificador.

Esse identificador é definido pelo programador quando ele declara a variável no código do programa.

E como criar um identificador?
Regras para criar um identificador:

    1. O início do identificador deve ser obrigatoriamente uma letra do alfabeto ou o caractere sublinhado (_), também conhecido como underscore;

    2. Os demais caracteres podem conter letras, números ou o caractere sublinhado (_);

    3. Não use espaços;

    4. Letras maiúsculas e minúsculas são diferentes:
      • Os indentificadores nomeados de Area e area referem-se as variáveis distintas.

    5. Não use uma palavra-chave reservada pela linguagem.

O tipo define a natureza dos dados que a variável armazena, e as operações que podem ser realizadas.

      Os tipos mais comuns no Python são:

    • Numérico, que ainda pode ser inteiro ou real;

    • Lógico; e

    • String de Caracteres.

Como são os tipos numéricos?
Tipos Numéricos


O tipo numérico no Python pode ser classificado de duas formas:

    • Inteiros (int) são números sem parte fracionária, tais como: 1 | 0 | -5 | 567

    • Reais (float) são números com parte fracionária, também conhecidos como ponto flutuante, tais como: 1.0 | 3.1415 | .75 | 23. | -2.7182

Observe que em Python, utilizamos o ponto para separar a parte inteira da parte fracionária de um número real.

A vírgula é usada para separar elementos de uma listagem, ou conjunto, como separar os argumentos da função print () 
# Ponto vs Vírgula:
print (1.5,15)

# Saída esperada:
1.5 15

Tipo Lógico

Uma variável do tipo lógico (ou boleano) assume apenas um entre dois valores possíveis:

    • True (verdadeiro)

    • False (falso)


Note que as iniciais T e F são escritas em maiúsculas.

String de Caracteres

Uma string (do inglês, corda) é uma cadeia de caracteres.

Ou seja, uma sequência de símbolos, tais como letras, números, sinais de pontuação, entre outros, que formam textos em geral.

Posso ver como fica no código?
Exemplos de String de Caracteres

#exemplos de String
texto = "Amo o Python."
print (texto)

texto_com_aspas = "Amo o \"Python\"."
print (texto_com_aspas)

print (texto.upper)
print (texto.lower)

#Saídas esperadas
Para declarar uma String usamos as aspas ("") para delimitar os caracteres que fazem parte de uma string.

Com isso é possível usar espaços dentro de uma string. E se for necessário usar o símbolo de aspas, usamos a expressão \".

Strings nos permitem também usar funções que podem ajustar todas as letras de uma string para maiúsculas (.upper()) ou minúsculas (.lower())

Papeis do Usuário e do Programador

A Entrada e Saída de Dados, para a programação, é a comunicação entre o Usuário e o Programador (ou o programa).

O Usuário é quem vai informar os dados necessários para a execução de um programa. O meio e essa informação são a Entrada de Dados

Da mesma forma, o usuário precisa dos resultados da execução desse programa. O meio e a informação que o usuário vai receber isso são a Saída de Dados

Então, cabe ao usuário inserir os dados de entrada e testar o programa ao observar as saídas de dados.

E ao programador cabe projetar o script desse programa. É o programador quem decide quando será necessario o usuário inserir ou receber dados através dos Comandos de Entrada e Saída.

E quanto as Expressões?












 Boas Práticas de Programação - Expressões 
 






Use espaços em branco para melhorar a legibilidade e utilize parênteses 
para melhorar a compreensão e evitar erros, mesmo que não altere a precedência 


Dos exemplos abaixo, qual você acha que é a melhor expressão?











B=(A**2+B**2)**0.5











B = ((A ** 2) + (B ** 2)) ** 0.5
